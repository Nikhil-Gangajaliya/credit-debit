1. Party Management
Add New Party: A dedicated module to register new contacts with their Name, Mobile Number, and Email Address.

Parties List: A dashboard view that displays all registered parties in a card format. Each card provides a quick glance at their contact info and current outstanding balance.

Dynamic Balance Tracking: The system automatically categorizes balances with status labels such as "To Collect" (Debit) or "Stable" (Zero balance).

Party Profiles: Capability to Edit or Delete party information at any time.

2. Transaction Recording
Debit & Credit Entry: A streamlined interface to record financial entries. Users can specify the Date, select a Party, define the Purpose (e.g., "Birthday"), and enter the amount.

Reference Tracking: Allows users to tag transactions with payment methods or reference IDs, such as RTGS, cash, or online transfers.

Transaction Editing: Each entry in a party's ledger can be individually updated or deleted if a mistake is made.

3. Comprehensive Reporting
The software includes robust reporting tools to visualize cash flow over time:

Party-wise Ledger: View a full history of transactions for a specific individual, including a running "Net" total.

Monthly Reports: A filterable view that aggregates all transactions for a specific month (e.g., October 2025).

Yearly Reports: A high-level overview of all financial activity within a chosen calendar year.

Data Export: All reports can be exported to PDF or Excel formats for external bookkeeping or sharing.
